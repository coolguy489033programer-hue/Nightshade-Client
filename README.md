# Nightshade-Client
Nightshade is a free, open-source Minecraft
hack client optimized for PvP and Bedwars.
If you get banned from Minecraft or a 
Minecraft server, don't blame me. I don't 
care what you choose to do with 
Nightshade; I'm just a random programmer,
and you are a person with free will, so 
use it responsibly—though I know you won't.
